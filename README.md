# Explore India – Tourism & Travel Discovery Platform

Explore India is a full-stack tourism platform developed for the NASA Space Apps Challenge. The application helps users discover Indian states, explore popular tourist destinations, view weather information, and gain travel insights through an interactive and user-friendly interface.

The platform combines tourism data, weather intelligence, and state-wise exploration features to provide travelers with a comprehensive travel planning experience.

---

## ✨ Features

### 🗺️ State-wise Exploration
- Explore all Indian states and union territories
- Discover famous tourist attractions
- Access state-specific travel information
- Interactive and user-friendly navigation

### 🌤 Weather Intelligence
- Real-time weather insights
- State-specific climate information
- Weather-based travel recommendations
- Dynamic weather visualization

### 🏛 Tourism Discovery
- Popular tourist destinations
- Cultural and historical attractions
- Regional travel information
- Travel planning assistance

### 📱 Responsive Design
- Mobile-friendly interface
- Modern user experience
- Responsive layouts
- Smooth navigation across devices

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/sebinjoseph005/explore-india.git
cd explore-india
```

### Install Dependencies

```bash
npm install
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm run dev
```
